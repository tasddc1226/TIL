# @staticmethod
> 정적 메서드에 대해서 알아보도록 하자.

## 형태

```Py
class 클래스이름:
    @staticmethod
    def 메서드(매개변수1, 매개변수2):
        코드
```

- 우선 정적 메서드는 클래스에서 바로 호출할 수 있는 메서드이다.
- 사용 방법은 함수위에 데코레이터를 사용해서 정적 메서드임을 알려줄 수 있다.


### 덧셈, 곱셈을 하는 정적 메서드
- 아래의 예시 코드를 보자

```Py
class Calculator:
    @staticmethod
    def add(a, b):
        print(a + b)

    @staticmethod
    def mul(a, b):
        print(a * b)

# 호출 방법
Calc.add(10, 20)
Calc.mul(10, 20)
```

- 정적 메소드는 클래스에서 바로 메서드를 호출하면 끝이다.
- 정적 메소드는 `self`를 인자로 받지 않으므로 인스턴스의 속성에는 접근할 수 없다!
- 사용하는 상황으로는, 인스턴스의 속성 그리고 인스턴스 메서드가 필요 없을 때 주로 사용한다.

## 그러면 무엇을 정적 메서드로 만들어야하나?

- 정적 메서드는 메서드의 실행이 외부 상태에 영향을 미치지 않는 순수 함수 즉, `pure function`을 만들 때 사용한다.
- 순수 함수는 부수 효과(side effect)가 없고 입력 값이 같으면 언제나 같은 출력 값을 반환한다!
- 따라서 `정적 메서드`는 인스턴스의 상태를 변화시키지 않는 메서드를 만들 때 사용한다!
